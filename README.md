# 🤖 Sentinel: Bionic Arm & SCADA System

Sentinel es un sistema de automatización industrial de bajo costo diseñado para la modernización de procesos en PYMES.

## 🚀 Tecnologías Clave
- **Microcontrolador:** ESP32 (C++ / Arduino).
- **Frontend/Backend:** Blazor Server (.NET 8).
- **Protocolos:** MQTT para comunicación en tiempo real.
- **Base de Datos:** SQLite para persistencia de trayectorias.
- **Sensores:** Fusión sensorial con Láser ToF (VL53L0X), Color (TCS34725) y Corriente (INA219).

## 🛠️ Innovación Mecánica
- Estructura híbrida con **base metálica** y refuerzos de **fibra de vidrio**.
- Diseño de **6 grados de libertad** optimizado para durabilidad industrial.

## 📊 Características
- Monitoreo en tiempo real (Telemetría).
- Mantenimiento predictivo basado en odometría digital.
- Interfaz Web HMI dinámica y segura.
